# leetcode-prep

## 45-Day DSA Preparation Plan

This plan is designed to help you prepare for Data Structures, Algorithms, and system design using a mix of Blind 75 patterns and recent LeetCode trends. Each day targets one LeetCode problem and one system design topic, with a checkbox to track progress.

### How to use this plan

- Practice one LeetCode problem and one system design topic each day.
- Review both the algorithmic pattern and the design trade-offs.
- Mark each day complete using the checkbox.
- Focus on consistency, clarity, and understanding the core pattern.

## Recommended Practice Tips

- Review one problem from each topic: arrays, strings, linked lists, trees, graphs, DP, and design.
- Keep notes for each pattern and write a short summary after solving.
- Revisit hard problems every weekend.
- Time yourself on at least two problems per week.

## Goal

By the end of 45 days, you should have covered the most common Blind 75 patterns and recent LeetCode trends, with a strong focus on problem-solving consistency and pattern recognition.
